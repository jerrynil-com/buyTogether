# Coding Alpaca 合購專案

## 如何使用？

在 terminal 輸入複製專案：

```
git clone https://github.com/codingalpaca/buyTogetherBase.git
```

進入專案：

```
cd buyTogetherBase
```

安裝所需的 gem：

```
bundle install
```

安裝前端套件：

```
yarn install
```

開啟伺服器：

```
rails server
```

如果你是在 C9 記得使用：

```
rails server -b $IP -p $PORT
```
